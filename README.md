CareerBridge is a modern Internship & Career Portal built to solve the biggest problems faced by students and professionals: spam opportunities, scattered platforms, irrelevant jobs, and lack of personalized career guidance.

It provides a unified dashboard where users can explore verified opportunities, discover skill gaps, follow a personalized roadmap, and track their career progress — all in one place.

🚀 Key Features

✨ What makes CareerBridge different:

🎯 Personalized & Verified Opportunities
AI-driven matching ensures you only see genuine and profile-fit internships.

📊 Skill Gap Insights & Career Roadmaps
Identify missing skills (High/Medium/Low priority) and follow step-by-step guidance.

📌 Unified Dashboard & Profile Management
Manage internships, insights, and your profile in one organized platform.

🔔 Progress Tracking & Notifications
Weekly growth reports and timely alerts so you never miss a deadline.

🛠 Tech Stack

Frontend: React.js + Vite + CSS

Backend (Planned): Node.js + Express

Database (Planned): MongoDB / PostgreSQL

AI/ML (Planned): Python (Scikit-learn, NLP)
